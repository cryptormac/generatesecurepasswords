Any password can be hacked in this day and age. There are plenty of ways hackers use to crack passwords. Two of the common ways are dictionary attack and brute force. Dictionary attack are all the words in a dictionary or list of passwords people use. Brute force is where the hacker will use every possible combination to figure what the password is.

Rules for creating passwords

1. Never use the names of your favourite friends, families toys or pets as passwords.

2. Never use dictionary words as passwords.

3. Never use identity numbers, birth dates, phone numbers, post codes or house addresses as passwords. Any hacker is bound to guess this right.

4. Never use passwords for more than one account, if a hacker hacks into one system and gains knowledge of the password it means the hacker can access any system that uses the same password.

5. A password should generally contain a minimum of 12 characters that should include a few of the following:

  1. Symbol - !@#$%^*(){}[]:";',.?/'"
  2. Small letter - abcdefghijklmnopqrstuvwxy
  3. Capital letter - ABCDEFGHIJKLMNOPQRSTUVWXY
  4. Number - 12345689

Secure your passwords all the time

Change your password every two months, due to the fact that if a hacker hacks into a web server where your MD5 hash value is stored then the chances of cracking the password is quite easy.

Always check the strength of your password, the stronger the password the longer it will take for the hacker to crack it.

Never store your password into any cloud application. ie: Google Drive, Dropbox, Gmail, Facebook, Twitter and so forth.

Manage your passwords with encryption. How this works is that you can encrypt a set passwords in plain text with a master password using encryption software.

Never log into a computer, mobile or tablet device that does not belong to you.

If you sell or give away any electronic device always make sure it is correctly formatted with all the data deleted. You do not want to give the electronic device away with stored passwords and device history.

Always lock your computer, mobile or tablet when not using them.

To keep your data secured on a hard drive, it is best to destroy it. This way you know nobody gets access to your information.

Keeping yourself safe with passwords on the internet

Do not save your password inside any web browser, anybody computer savvy person is able to read the password stored inside the browser.

Where possible try to use dual authentication to access any web application.

Never log into important account information using a wi-fi hotspot.

When using a wi-fi hotspot, always ensure that you use VPN.

Never send passwords via text and instant messaging such as Whatsapp, Facebook messenger, Twitter Direct Messaging and so forth.

Only use internet websites that are secure, if the website does not have a secure sign on the top right and you enter a password, a hacker is able to intercept the website and gain access to your password.

Ensure that you are accessing the website directly and not a scam or phishing site. If are accessed the wrong website with wrong url and typed your password in, this means the hacker or person owning the web hosting has your password. Make sure you change it immediately.

Always have anti-virus installed on your computing system.

Have two web browsers - one two access important information and the other to access non important information.

Never reset your password from the internet unless you know for sure that you requested it to be reset.
